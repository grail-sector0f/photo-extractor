    **Step 5: Create lib/keepalive.ts**

    export async function keepAlive(): Promise<void> {

    }

    **Step 6: Create lib/download.ts**

    Export two functions:

    `buildSafeFilename(basename: string, ext: string, subfolder: string): Promise<string>`
    - Query `chrome.downloads.search({ filenameContains: basename, limit: 100 })` to find existing matches
    - Filter results: only those whose `filename` property contains the full `${subfolder}/${basename}` path segment (search is substring-based, so filter precisely)
    - Build a Set of taken filenames from the results
    - If `${subfolder}/${basename}.${ext}` is NOT in the Set, return it (clean name, no collision)
    - Otherwise loop i from 1 to 99, build candidate `${subfolder}/${basename}_${String(i).padStart(2, '0')}.${ext}`, return first candidate NOT in the Set
    - Throw Error if all 99 slots exhausted

    `triggerDownload(url: string, basename: string, ext: string): Promise<number>`
    - Call `keepAlive()` first
    - Call `buildSafeFilename(basename, ext, 'travel-photos')` to get the safe filename
    - Call `chrome.downloads.download({ url, filename: safeFilename, conflictAction: 'overwrite', saveAs: false })`
    - Return the download ID from the callback/promise
    - The `conflictAction: 'overwrite'` is safe because we already handle collision avoidance ourselves

    **Step 7: Create entrypoints/background.ts**

    ```typescript
    import { triggerDownload } from '@/lib/download';

    export default defineBackground({
      main() {
        chrome.runtime.onInstalled.addListener(() => {
          console.log('[photo-extractor] extension installed');
        });

        chrome.runtime.onMessage.addListener((message, _sender, sendResponse) => {
          if (message.type === 'DOWNLOAD_FILE') {
            const { url, basename, ext } = message.payload;
            triggerDownload(url, basename, ext)
              .then((downloadId) => sendResponse({ ok: true, downloadId }))
              .catch((err) => sendResponse({ ok: false, error: String(err) }));
            return true; // keep message port open for async response
          }
        });
      },
    });
    ```
