# Phase 999.3 Plan 02: Library UI Summary

**FilterBar library view with per-record delete, clear-history button, and CDN hotel-path guard — all verified working in Chrome**

- **Duration:** ~45 min (includes human verification round-trip)

- **Completed:** 2026-03-22T16:15:00Z

## Accomplishments

- Library view renders with FilterBar (Destination, Category, Year dropdowns), record list with thumbnail + tag chips + trash button, empty state, and 3-row loading skeletons
- filterRecords() and compareRecords() exported as pure functions with full unit test coverage (187 tests passing)
- removeFromLibrary() added for per-record deletion; ERASE_DOWNLOADS clears Chrome's download manager list
- CDN rewrite bug fixed — Booking.com /xphoto/ attraction paths no longer 404 because rewrite now guards on /hotel/ in pathname
- Required field asterisks and improved button copy ("Fill in required fields") improve form guidance

## Files Created/Modified

- `entrypoints/popup/App.tsx` - LibraryPanel, FilterBar, LibraryRecord (with trash), EmptyLibraryState, LibrarySkeletons; filterRecords() and compareRecords() exported; DOWNLOAD_RESET action; StatusMessage clear button; required field asterisks
- `lib/library.ts` - removeFromLibrary() added
- `lib/cdnRewrite.ts` - rewriteBooking() now guards on /hotel/ pathname to skip /xphoto/ attraction images
- `entrypoints/background.ts` - ERASE_DOWNLOADS message handler using chrome.downloads.erase
- `tests/unit/library-sort.test.ts` - Unit tests for filterRecords() and compareRecords()

## Decisions Made

- SortBar replaced with FilterBar at user request during verification — filter-by-field is more useful than sort-by-field for Jennifer's use case
- compareRecords() simplified to savedAt-only; sort order is always newest-first (not user-configurable)
- removeFromLibrary() scoped to chrome.storage.local only — does not touch files on disk
- CDN rewrite scoped to /hotel/ paths because /xphoto/ (attractions) returned 404 at max10000x10000; leave xphoto at native resolution

**3. [User Direction] Clear download history button**
- **Found during:** Task 2 (human verification)
