    2. Create `lib/extract/srcsetParser.ts` implementing `parseSrcset(srcset: string): string | null`:
       - Trim input; return null if empty
       - Split on commas, trim each candidate
       - For each candidate: split on whitespace, first token is URL, second is descriptor
       - Descriptors ending in `w`: parse as width number
       - Descriptors ending in `x`: parse as density number
       - No descriptor: treat as `x: 1`
       - If any `w` candidates exist, return the one with largest `w` value
       - Otherwise if any `x` candidates exist, return the one with largest `x` value
       - Return null if no valid candidates parsed
