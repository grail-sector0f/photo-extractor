    1. `rewriteBooking(url: URL): string` -- hostname ends with `bstatic.com`. Replace `/max\d+(x\d+)?/` in pathname with `/max10000x10000/`. Preserve all query params (k=, o= are auth tokens). If no match in pathname, return url.href unchanged.

    2. `rewriteAirbnb(url: URL): string` -- hostname ends with `muscache.com`. If `url.searchParams.has('aki_policy')`, set to `xx_large`. Else if `url.searchParams.has('im_w')`, set to `1440`. Else return url.href unchanged (no size param to rewrite).

    3. `rewriteCloudinary(url: URL): string` -- hostname is exactly `res.cloudinary.com`. Replace the first transformation segment after `/upload/` using regex `/\/upload\/[^/]+\//` with `/upload/w_4000,c_limit,q_auto/`. If no `/upload/` in path, return url.href unchanged.

    4. `rewriteImgix(url: URL): string` -- hostname ends with `.imgix.net` (note the leading dot -- prevents matching `notimgix.net`). Set `w=4000`, delete `h`, set `fit=max`. Return url.href.

    5. `rewriteViator(url: URL): string` -- hostname ends with `dynamic-media-cdn.tripadvisor.com`. Set `w=4000`, set `h=-1` (unconstrained height -- this is NOT a bug, it is documented in Viator Partner API). Preserve `s` param. Return url.href.

    6. `rewriteGetYourGuide(url: URL): string` -- hostname is exactly `cdn.getyourguide.com`. Return url.href unchanged. Add TODO comment: "GetYourGuide resize params unknown -- inspect live site URLs in DevTools before implementing."
