    **sendDownloadMessage function** (from RESEARCH.md code example):
    ```typescript
    function sendDownloadMessage(url: string, basename: string, ext: string): Promise<void> {
      return new Promise((resolve, reject) => {
        chrome.runtime.sendMessage(
          { type: 'DOWNLOAD_FILE', payload: { url, basename, ext } },
          (response) => {
            if (chrome.runtime.lastError) {
              reject(new Error(chrome.runtime.lastError.message));
              return;
            }
            if (response?.ok) resolve();
            else reject(new Error(response?.error ?? 'unknown error'));
          }
        );
      });
    }
    ```

    **runDownloads function** (from RESEARCH.md Pattern 3):

    - On completion: `dispatch({ type: 'DOWNLOAD_DONE', saved, failed })`
    - After successful download: persist fields to `chrome.storage.local.set({ destination, vendor, category })`
