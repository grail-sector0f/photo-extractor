    Create `lib/naming.ts` with three exported functions:

  1. `normalizeField(value: string): string`
    - `.trim()` first
    - `.toLowerCase()`
    - `.replace(/\s+/g, '-')` (spaces to hyphens)
    - `.replace(/[^a-z0-9\-]/g, '')` (strip anything not alphanumeric or hyphen)

  1. `buildBasename(destination: string, vendor: string, category: string, notes?: string): string`
    - Apply `normalizeField()` to destination, vendor, category
    - If `notes` is provided and `notes.trim()` is non-empty, apply `normalizeField(notes)` and append as 4th segment
    - Join all segments with underscore `_`
    - Return the joined string

  1. `deriveExt(url: string): string`
    - Try `new URL(url).pathname`
    - Find last `.` in pathname
    - If no dot, return `'jpg'`
    - Extract extension after dot, lowercase it
    - Accept only: `['jpg', 'jpeg', 'png', 'webp', 'gif', 'avif']`
    - Return the extension if known, otherwise `'jpg'`
    - Wrap in try/catch — return `'jpg'` on any error
