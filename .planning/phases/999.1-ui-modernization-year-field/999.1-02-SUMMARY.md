## What Was Built

Full Material Design 3 design system foundation:

**Color tokens** — 20 MD3 custom colors added to `tailwind.config.js` covering primary, secondary, tertiary, surface hierarchy, outline, and error. All usable as Tailwind utility classes (`bg-primary-container`, `text-on-surface-variant`, `ring-primary`, etc.).

**Typography** — `font-manrope` and `font-inter` utility classes added. Both fonts bundled as WOFF2 files in `public/fonts/` — no CDN dependency required, which is essential since Google Fonts CDN is inaccessible from extension popups.

**Border radius** — MD3 shape scale overrides (`rounded-lg` = 0.5rem, `rounded-xl` = 0.75rem, `rounded-2xl` = 1rem).

**Font loading** — `@font-face` declarations in `style.css` use variable weight ranges (400-800 for Manrope, 400-700 for Inter) so a single file covers all weights. The `/fonts/` URL path works because WXT copies `public/` contents to the build output root.

**Build verification** — `npx wxt build` confirmed both font files appear in `.output/chrome-mv3/fonts/` (24KB Manrope, 48KB Inter).
